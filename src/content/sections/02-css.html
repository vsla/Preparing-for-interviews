2. CSS (Deep) 10 topics

CSS questions test both theoretical knowledge and practical problem-solving. Interviewers probe the box model, specificity, layout systems (flexbox and grid), stacking contexts, and responsive design. A solid answer always includes edge cases and browser quirks, not just the happy path.

2.1 The Box Model 
/* content-box (default): width/height = content only */
.box { width: 200px; padding: 20px; border: 5px solid; }
/* Rendered width: 200 + 40 + 10 = 250px */

/* border-box: width/height includes padding + border */
.box { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid; }
/* Rendered width: 200px (content = 200 - 40 - 10 = 150px) */

/* Global border-box strategy */
*, *::before, *::after { box-sizing: border-box; }

Margin Collapse

/* Margins collapse between adjacent block siblings */
.a { margin-bottom: 20px; }
.b { margin-top: 30px; }
/* Gap = max(20, 30) = 30px, not 50px */

/* Does NOT collapse when:
   - Elements have padding or border between them
   - Parent has overflow != visible
   - Elements are flex/grid children
   - Absolute/fixed positioning
*/

/* Preventing collapse with padding */
.parent { padding-top: 1px; } /* forces parent-child margins apart */

/* Parent-child collapse */
.parent { margin-top: 10px; }
.child  { margin-top: 20px; }
/* If no border/padding on parent: only 20px gap from viewport */

Block Formatting Context (BFC)

/* BFC created by: */
overflow: hidden; /* most common trick */
display: flow-root; /* modern, purpose-built */
position: absolute | fixed;
display: flex | grid;
float: left | right;

/* BFC contains floats and prevents margin collapse with children */
.clearfix { overflow: hidden; } /* old way */
.clearfix { display: flow-root; } /* modern way */
2.3 Flexbox (Deep)

Container Properties

.container {
  display: flex;          /* or inline-flex */
  flex-direction: row;    /* row | row-reverse | column | column-reverse */
  flex-wrap: nowrap;      /* nowrap | wrap | wrap-reverse */
  flex-flow: row nowrap;  /* shorthand for direction + wrap */

  /* Main axis alignment */
  justify-content: flex-start; /* flex-start|flex-end|center|
                                   space-between|space-around|space-evenly */

  /* Cross axis alignment (all items) */
  align-items: stretch;   /* stretch|flex-start|flex-end|center|baseline */

  /* Multi-line cross axis alignment */
  align-content: normal;  /* (only when flex-wrap: wrap) same values */

  gap: 1rem;             /* row-gap and column-gap */
  row-gap: 0.5rem;
  column-gap: 1rem;
}

Item Properties

.item {
  flex-grow: 0;   /* how much to grow relative to siblings (unitless) */
  flex-shrink: 1; /* how much to shrink (default 1) */
  flex-basis: auto; /* initial size before grow/shrink */
  flex: 1;        /* shorthand: grow=1 shrink=1 basis=0 */
  /* flex: none = 0 0 auto (no grow, no shrink) */
  /* flex: auto = 1 1 auto */

  align-self: auto; /* overrides align-items for this item */
  order: 0;       /* visual order, not DOM order */
  min-width: 0;   /* fix for overflow with flex: 1 */
}

Common Patterns

/* Perfectly centered div */
.parent { display: flex; justify-content: center; align-items: center; }

/* Sticky footer */
body { display: flex; flex-direction: column; min-height: 100vh; }
main { flex: 1; } /* pushes footer down */

/* Holy Grail layout */
body { display: flex; flex-direction: column; min-height: 100vh; }
.content-area { display: flex; flex: 1; }
aside { width: 200px; flex-shrink: 0; }
main  { flex: 1; }

/* Equal-width columns */
.col { flex: 1; min-width: 0; } /* min-width:0 prevents overflow */

/* Responsive nav: items take available space */
.nav-item { flex: 1; text-align: center; }
.nav-logo  { flex: 0 0 auto; } /* fixed size logo */
2.4 Grid (Deep) 
.grid {
  display: grid;

  /* Explicit tracks */
  grid-template-columns: 200px 1fr 2fr;
  grid-template-rows: auto 1fr auto;

  /* Repeat notation */
  grid-template-columns: repeat(3, 1fr);
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

  /* Named areas */
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";

  gap: 1rem;
  grid-auto-rows: minmax(100px, auto);    /* implicit row sizing */
  grid-auto-columns: 1fr;
  grid-auto-flow: row dense; /* dense packs holes */
}

/* Placing items */
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }

/* Explicit placement */
.item {
  grid-column: 1 / 3;       /* span columns 1 to 3 */
  grid-column: span 2;      /* span 2 columns */
  grid-row: 2 / 4;
  grid-column: 1 / -1;      /* full width (last line = -1) */
}

/* Subgrid — nested grid inherits parent tracks */
.card { display: grid; grid-row: span 3; grid-template-rows: subgrid; }

Grid vs Flexbox

Use Grid whenUse Flexbox when
2D layout (rows AND columns)1D layout (row OR column)
Layout-first designContent-first design
Overlapping elementsAlignment within a row/col
Complex page layoutsNavigation bars, toolbars
Precise grid alignmentCentering a single element
2.5 Positioning 
/* static: default, top/left/right/bottom/z-index ignored */
/* relative: offset from normal position, still in flow */
/* absolute: removed from flow, positioned relative to nearest positioned ancestor */
/* fixed: removed from flow, positioned relative to viewport */
/* sticky: relative until scroll threshold, then fixed-like */

.modal-overlay { position: fixed; inset: 0; } /* inset = top+right+bottom+left: 0 */

/* Absolute needs a positioned ancestor */
.parent   { position: relative; }  /* creates positioning context */
.tooltip  { position: absolute; top: 100%; left: 0; }

/* Sticky gotchas */
.sticky-header { position: sticky; top: 0; }
/* Fails if any ancestor has: overflow: hidden|auto|scroll */
/* Must have a defined height on the scrollable ancestor */

/* z-index and stacking contexts */
/* New stacking context created by: */
/* - position != static + z-index != auto */
/* - opacity < 1 */
/* - transform, filter, will-change, isolation: isolate */
/* - position: fixed | sticky */

/* z-index only works within the same stacking context! */
.parent { position: relative; z-index: 1; }
.child  { position: absolute; z-index: 9999; } /* can't escape parent's z-index */

Specificity Calculation

Format: (inline, IDs, classes/attrs/pseudoclasses, elements/pseudoelements)

Cascade Layers (@layer)

/* Lower layers lose to higher layers, regardless of specificity */
@layer reset, base, components, utilities;

@layer reset {
  * { margin: 0; padding: 0; }
}
@layer components {
  .btn { color: blue; } /* overrides reset even with lower specificity */
}
@layer utilities {
  .text-red { color: red; } /* wins over components */
}

/* Unlayered styles win over all layers */

CSS Custom Properties

:root {
  --color-primary: #1a73e8;
  --spacing-md: 1rem;
  --border-radius: 4px;
}

.button {
  background: var(--color-primary);
  padding: var(--spacing-md);
  border-radius: var(--border-radius);
}

/* Fallback value */
color: var(--undefined-var, red);

/* Scoped custom properties */
.dark-theme { --color-primary: #90caf9; }

/* Computed at runtime — can be changed with JS */
document.documentElement.style.setProperty("--color-primary", "#ff0000");
2.7 Responsive Design 
/* Mobile-first: min-width breakpoints */
.container { padding: 1rem; } /* mobile default */
@media (min-width: 768px)  { .container { padding: 2rem; } }
@media (min-width: 1024px) { .container { max-width: 960px; margin: auto; } }

/* Viewport units */
100vw  /* viewport width */
100vh  /* viewport height — problematic on mobile (browser chrome) */
100dvh /* dynamic viewport height — accounts for mobile browser chrome */
100svh /* small viewport height */
100lvh /* large viewport height */

/* Fluid typography */
font-size: clamp(1rem, 2.5vw, 2rem);
/* min=1rem, preferred=2.5vw, max=2rem */

width: min(90%, 1200px);   /* take 90% but never more than 1200px */
padding: max(1rem, 2vw);   /* at least 1rem */

/* Container queries — respond to container size, not viewport */
.card-container { container-type: inline-size; }

@container (min-width: 400px) {
  .card { display: flex; flex-direction: row; }
}

/* aspect-ratio */
.video { aspect-ratio: 16 / 9; width: 100%; }
.avatar { aspect-ratio: 1; width: 3rem; }

/* object-fit for images */
img { width: 100%; height: 200px; object-fit: cover; object-position: top; }
2.8 Animations & Transitions 
/* Transitions */
.btn {
  transition: background-color 200ms ease, transform 150ms ease-out;
  /* property duration timing-function delay */
}
.btn:hover {
  background-color: darkblue;
  transform: translateY(-2px);
}

/* Keyframe animation */
@keyframes slideIn {
  from { transform: translateX(-100%); opacity: 0; }
  to   { transform: translateX(0);     opacity: 1; }
}
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50%       { transform: translateY(-20px); }
}

.modal {
  animation: slideIn 300ms ease-out forwards;
  /* name duration timing fill-mode */
  /* fill-mode: forwards = keep end state after animation */
}

/* GPU-accelerated properties (compositor layer) */
/* transform, opacity → no reflow/repaint */
/* left, top, width, height → triggers layout */

/* will-change: hint to browser to create GPU layer */
.animated { will-change: transform; }
/* Don't overuse — creates new stacking context, uses GPU memory */

/* Reflow (layout) → Repaint → Composite */
/* Most expensive: JS → Style → Layout → Paint → Composite */
/* Cheapest animation: only composite (transform, opacity) */

/* Respecting user preferences */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}

2.10 Tricky CSS Interview Questions

Common gotchas

/* Q: Why doesn't vertical margin work on inline elements? */
/* A: Inline elements respect left/right margin but top/bottom
      doesn't affect surrounding elements — use line-height */

/* Q: What is a stacking context and what creates one? */
/* A: Isolated z-index group. Created by: opacity < 1,
      transforms, filters, position+z-index, will-change,
      isolation: isolate, fixed/sticky */

/* Q: Why does my absolute element not position relative to parent? */
/* A: Parent needs position != static (relative/absolute/fixed/sticky) */

/* Q: Why doesn't sticky work? */
/* A: Check for overflow:hidden on any ancestor */
/* A: Element must have top/left/right/bottom set */
/* A: Parent must be taller than the sticky element */

/* Q: Specificity of :not() */
/* A: :not() itself has 0 specificity, but its argument counts */
div:not(.special)  → 0,0,1,1 (class + element)
*:not(p)           → 0,0,0,1 (just element)

⚡ Key Takeaways

  • box-sizing: border-box makes width/height include padding and border — apply it globally in resets
  • Margins collapse between adjacent block siblings; the larger wins (does not apply to flex/grid children)
  • Flexbox is one-dimensional (row or column); Grid is two-dimensional — choose accordingly
  • Specificity order: inline styles > IDs > classes/attributes/pseudo-classes > elements
  • position: sticky requires a defined top/left value and a scrollable parent — overflow: hidden on parent breaks it
  • transform and opacity only trigger compositing — much cheaper than layout-triggering properties
  • Container queries (@container) are more useful than media queries for component-level responsiveness
  • CSS custom properties (--var) cascade and inherit, making them the backbone of modern theming
